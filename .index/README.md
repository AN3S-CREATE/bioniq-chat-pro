# Project Context Index (.index/)

This directory maintains the living context of the Bioniq Chat Pro standalone application migration project. It helps developers and AI agents quickly understand the project's state, files, architecture, and key decisions.

## Contents

- `file-inventory.md` - Structured inventory of files, their purposes, and status.
- `architecture.md` - High-level system overview, component flows, database schemas, and AI orchestration.
- `key-decisions.md` - Record of architectural and process decisions.
- `dead-code.md` - Log of legacy/unused/dead code or technical debt.
- `context-refresh-log.md` - Record of context refreshes and index updates.

## Maintenance Rules

1. Update the index whenever a file is created, modified, moved, or deleted.
2. Keep descriptions concise and accurate.
3. Review and update the architecture when design decisions or flows change.
